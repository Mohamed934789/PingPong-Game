## ğŸ§  How to Play

- **Left Player** controls:
  - `W` = Move paddle up
  - `S` = Move paddle down

- **Right Player** controls:
  - `Up Arrow` = Move paddle up
  - `Down Arrow` = Move paddle down

- First player to reach **5 points** wins the game.
- After winning, the game resets automatically after 2 seconds.
